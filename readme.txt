If could not browse the web properly, please follow the steps below.

Firefox:

1. Enter "about:config" in the address bar of the Firefox browser and press Enter.

2. Search for "security.fileuri.strict_origin_policy" in the filter.

3. Set security.fileuri.strict_origin_policy to false

4. Close all Firefox windows currently open and restart Firefox.


If using Chrome, please refer to the website below：

http://www.chrome-allow-file-access-from-file.com/windows.html
